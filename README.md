# text-pad
A simple auto save text pad with sqlite

[Download](https://github.com/sukesh-s/text-pad/files/8885408/HelloWorld.zip)
Folder includes 
- Install and Uninstall bash script
- Desktop file 
- App database
- App icon

To run
```
java -jar Helloworld.jar
```
